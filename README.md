# Chat em tempo real

Este projeto é o front-end do **chat em tempo real** desenvolvido em **Next JS** com **Tailwind CSS**.  
Ele permite você conversar com outras pessoas em tempo real quando integrado com seu back-end.

---

## Tecnologias usadas

- **Next JS** – Servidor web.
- **Tailwind CSS** - Usado para estilizar a página.

---

## Versão online do projeto:

Caso você queira ver o projeto em execução de forma mais rápida e facíl acesse o link: https://chat.maycon.dev.br/

---

## Requerimentos

- **Node.js** – A partir da versão 22.
- **npm** – A partir da versão 10.

---

## Variáveis de ambiente (.env)

```bash
URL_WEBSOCKET=ws://localhost:8080
```

---

## Passo a passo para rodar localmente

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/Maycon40/chat-tempo-real.git
cd chat-tempo-real
```

### 2️⃣ Instalar as dependências

```bash
npm i
```

### 3️⃣ Subir o servidor web

```bash
npm run dev
```

---

## Observações

- O back-end precisa está rodando para que o chat funcione.
- O back-end do chat está no sequinte repositório: https://github.com/Maycon40/back-end-chat-tempo-real.
